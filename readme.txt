Para clonar debes ejecutar
git clone https://github.com/Carlos-Mancilla-develop/demo_mi_pagina_personal.git

Se está usando Bootstrap 5.3.3 y JQuery 3.7.1.

El nombre de archivo a subir a la plataforma debe seguir el siguiente formato:
consolidacionX-nombre-alumno.zip

Donde X representa el número de módulo de la consolidación.
Ej:
consolidacion2-Juan-Perez.zip
